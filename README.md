# Mobile Phone Evaluation System

## Overview
The Mobile Phone Evaluation System is a machine learning-based application that predicts and evaluates mobile phone prices based on their specifications. The system uses a trained model to analyze features such as RAM, battery capacity, display size, camera specifications, and processor type.

## Features
- Mobile price prediction using Machine Learning
- User-friendly interface
- Feature preprocessing and encoding
- Real-time predictions
- Lightweight deployment

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle

## Installation

### Clone the Repository

```bash
git clone https://github.com/Anas-tp/Mobile-Phone-Evaluation-System.git
```

### Navigate to the Project Folder

```bash
cd Mobile-Phone-Evaluation-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```
